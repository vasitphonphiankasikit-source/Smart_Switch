<h1>ระบบควบคุมสวิตช์ไฟอัตโนมัติด้วยเซนเซอร์ตรวจจับแสง</h1>
<img src="projectSmartSwitch.JPG" width="auto" height=300px></img>
ผมสังเกตว่าในชีวิตประจำวัน ในบางวันที่ฟ้าเริ่มมืด แต่ไฟหน้าบ้านก็ยังไม่ติด และบางวันยังมีแสงสว่างอยู่ ไฟหน้าบ้านก็ติดแล้ว ทำให้ต้องเสียเวลาตั้งเวลาเปิดไฟ แต่ไฟหน้าบ้านยังคงต้องปิดตอนเที่ยงคืนอัตโนมัติทุกวัน จึงแก้ปัญหา และประหยัดพลังงาน
<h2>อุปกรณ์ที่ใช้:</h2>
<ul>
  <li>เซนเซอร์ตรวจจับแสง</li>
  <li>รีเลย์ควบคุมไฟฟ้า</li>
  <li>ESP32 Board</li>
  <li>ปุ่มกด</li>
  <li>สายไฟและหลอดไฟRGB</li> 
  <li>หลอดไฟ LED RGB</li>
</ul>
<img src="diagram.png" width="auto" height=300px></img>
<h2>หลักการทำงาน: </h2>
 โดยเริ่มต้น บอร์ดจะหาสัญญาณ Wi-Fi เพื่อตั้งเวลาอัตโนมัติ โดยหลอดไฟ LED RGB จะกะพริบเป็นสีเหลือง จนกว่าจะมีสัญญาณ Wi-Fi เมื่อมีแสงสว่างจากดวงอาทิตย์ที่น้อยลง ระบบจะสั่งให้รีเลย์เปิดไฟโดยอัตโนมัติและให้แสดงสถานะ หลอดไฟ LED RGB เป็นสีเขียว หากเป็นเวลาที่ตั้งกำหนดไว้ คือช่วงเวลาเที่ยงคืนของทุกวัน ไฟจะปิดอัตโนมัติ หรือมีการกดปุ่ม ระบบจะสั่งให้ปิดไฟเองโดยใช้สวิตช์มือ และเปลี่ยนสถานะ หลอดไฟ LED RGB เป็นสีแดง 
<img src="circuit.JPG" width="auto" height=300px></img>
<h2>สิ่งที่ได้เรียนรู้:</h2>
ได้ทักษะฝึกการเขียนโค้ด Arduino ไม่ว่าจะเป็น การต่อวงจรไฟฟ้า และการแก้ไขปัญหาเฉพาะหน้า เช่น การปรับค่าความไวของเซนเซอร์ให้เหมาะสมกับพื้นที่ใช้งาน หรือสามารถปรับเปลี่ยนเวลาให้ปิดไฟอัตโนมัติในเวลาที่ต้องการได้
